Directions to run this program:

1. Clone directory

  ```
  git clone https://github.com/nss-day-cohort-38/welcome-to-nashville-nimble-ants.git
  ```
2. Open cloned folder

```
cd welcome-to-nashville-nimble-ants/
```

3. Run https-server

```
hs -o src/index.html
```

1. Ticketmaster API
2. Zomato API
